# 📊 Sales Performance Dashboard (Microsoft Excel)

## 📌 Project Overview

This project is an interactive Sales Performance Dashboard developed in Microsoft Excel to analyze sales data across multiple cities and sales executives. The dashboard enables users to filter data dynamically using slicers and gain meaningful business insights through charts and Pivot Tables.

---

## 🎯 Objective

The objective of this project is to transform raw sales data into an easy-to-understand interactive dashboard that helps monitor sales performance, target achievement, and executive-wise analysis.

---

## ✨ Features

- Interactive City Slicer
- Executive-wise Sales Analysis
- Total Sales Comparison
- Target Hit Percentage
- Away From Target Percentage
- Dynamic Pivot Tables
- Interactive Pivot Charts
- User-friendly Dashboard Layout
